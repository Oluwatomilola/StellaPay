# Stellar Simple Payment dApp

A beginner-friendly decentralized application that allows users to connect their Freighter wallet, view their XLM balance, and send XLM transactions on the Stellar testnet.

## Features
- Connect / disconnect Freighter wallet
- Fetch and display XLM balance
- Send XLM to any Stellar address
- Transaction success / error feedback
- Testnet explorer link

## Tech Stack
- React + TypeScript
- Stellar SDK
- Freighter API
- Vite

## Prerequisites
- Node.js >= 18
- Freighter wallet (Testnet)

## Setup Instructions

```bash
git clone https://github.com/<your-username>/stellar-simple-pay
cd stellar-simple-pay
npm install
npm run dev
